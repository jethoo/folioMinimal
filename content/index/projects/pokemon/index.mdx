React Application for finding your favorite pokemon. Data for the names, types and points of pokemon is coming from an api. User can type in the search field which triggers the display to show the names and types of pokemon matching the entered character. Image of the pokemon is also shown. And also the matching character is highlighted in the display. 
Pokemons can be sorted by Maximum Combat Points by clicking the checkbox. Only top 4 matches are displayed.
